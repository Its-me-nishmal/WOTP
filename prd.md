# PRODUCT REQUIREMENTS DOCUMENT — WOTP

Product Name:
WOTP (WhatsApp One-Time Password Platform)

Problem:
SMS OTP is expensive and unreliable.

Solution:
WhatsApp-based OTP & Messaging API.

Target Users:
Developers, startups, SaaS products.

Core Features:
- Google Login
- WhatsApp QR Connect
- API Key generation
- Send OTP
- Verify OTP
- Logs
- Dashboard

Free Plan:
100 OTP/month

Pro Plan:
10,000 OTP/month

Security:
- OTP expiry
- Hash storage
- Rate limits

Success Criteria:
User signs up → connects WhatsApp → sends OTP → verifies OTP.

Out of Scope (v1):
Marketing campaigns
Bulk blasting
Voice calls
